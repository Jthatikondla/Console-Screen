Project on Console

Visual Studio is a place perfect for typing code. It helps beginners type faster it also helps advanced coders. You can set up visual sudia with 3-4 easy steps. The first step is downloading Visual Studio IDE software. Then create a folder in your files. Open Visual Studio and click File > Open Folder, then browse and open the folder that you just created. The last step is to Install the Live-Server Exstension in Visual Studio.

Console screen is accesable from right clicking on the computer and clicking on inspect. Then click Console on the top. You can print anything on console by tyipne console.log() in your code.
